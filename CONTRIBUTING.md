# Contributing to VBOX

Thank you for your interest in contributing to **VBOX**!  
This guide will help you set up the project, understand the workspace structure,
and contribute smoothly.

---

## Clone the repo

`git clone git@github.com:timothyokooboh/vbox.git`

## Install & build

```bash
pnpm install
pnpm build
```

## Start the documentation

```bash
    cd apps/docs
    pnpm dev
```

## Code of Conduct

Please be respectful and constructive when interacting with others. Help keep
the project welcoming for contributors of all experience levels.
